# outputs/

This folder stores all artefacts generated by the notebook at runtime — charts, evaluation tables, and any exported result files.

> **This folder is mostly empty in the repository.** All files inside are produced when you run `notebook/group_submission.ipynb` from top to bottom. Re-running the notebook will regenerate them.

---

## Subfolder: `charts/`

Six PNG charts are saved here by the notebook, one per visualisation cell:

| File | Section | Description |
|------|---------|-------------|
| `top30_ingredients.png` | §2.2 | Horizontal bar chart of the 30 most frequent ingredients across all 231,637 recipes. Highlights the dominance of common pantry staples (salt, butter, sugar, flour). |
| `ingredients_per_recipe.png` | §2.3 | Histogram of the number of ingredients per recipe. Median ≈ 9; distribution is right-skewed with a long tail toward 30+ ingredients. |
| `rating_distribution.png` | §2.4 | Bar chart of rating counts (1–6) with percentage annotations. Shows that 73% of reviews give the maximum rating of 6. |
| `calorie_distribution.png` | §2.5 | Histogram of calories per recipe (clipped at 5,000 kcal). Median ≈ 313 kcal; strong right skew from bulk or party-sized recipes. |
| `precision_recall_at_k.png` | §4.3 | Line chart of Precision@K, Recall@K, and F1@K on the validation set for K ∈ {5, 10, 15, 20, 25}. Used to choose the best K. |
| `ingredient_substitution.png` | §6.2 | Three horizontal bar charts showing cosine-similarity scores for the top-5 substitutes of `butter`, `chicken`, and `flour`. |
| `test_evaluation.png` | §7.2 | Grouped bar chart comparing Precision@K, Recall@K, and F1@K on the held-out test set at K=5 and K=10. |
| `diversity_distribution.png` | §7.3 | Histogram of within-user recommendation diversity scores (average pairwise 1 − cosine) across all 1,701 test users. |
| `calorie_aware_comparison.png` | §7.4 | Side-by-side horizontal bar charts comparing the Top-10 ingredient list before and after calorie-aware re-ranking for a sample user. |

## Evaluation Summary (test set)

| Metric | K=5 | K=10 |
|--------|-----|------|
| Precision@K | 0.5848 | 0.5032 |
| Recall@K | 0.0674 | 0.1150 |
| F1@K | 0.1208 | 0.1872 |

These results are from the held-out test set of 1,701 cold-start users (users never seen during training). Validation results (P@10 = 0.502, R@10 = 0.117) closely match, confirming the model generalises consistently.
